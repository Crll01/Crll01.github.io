Accessory Architecture #

Eyeglass #

準則：

1. 材質

金屬、板材還是玳瑁。材質決定整體質感和風格。

2. 鼻托

亞洲人最好選有鼻托的款式，除非臉部特別立體。

鼻托高度也關鍵，如果鼻托太低會讓鏡框貼臉，最好改成高鼻托才顯得臉立體。市面上一般不夠，要找人手動改。

3. 形狀

直角曲度光譜：四角方形 👉 全圓角，這幾種之間要根據想表現的風格來選。（前提是不要太醜）

4. 顏色

金屬框一般從黑到灰是主流，彩色就看個人適配。

比方說Y2K？紫色感覺就是溫柔女教師…

5. 總結

選眼鏡先看材質，再看鼻托結構，然後決定形狀與灰度；整體取決於你想呈現的風格與立體度。

僅限模板使用，特定情況除外。

另外：風格化取決於個人情況

A : 板材感 + 有鼻托 + 不贴脸 -> 日產手作 e.g. Masunaga、EYEVAN、金子眼镜 （要找代購）

General accessories #

  1. 寶石，就是珍珠、歐泊之類的，那種是定制的。

  2. 潮流/設計師，裝飾性比較強。（例如VW / MM6

  3. 先鋒，類似於OLIO E ACETO，在買手店裡出現的那種。

  4. 古著，要看具體類型。像古著的話，買鑽石會有老礦鑽石，古著風格會比較特別，比如維多利亞那種。

  5. …高橋吾郎和他的複製品？😂

  6. 有一種另類的的好東西，玉雕或者水晶雕刻牌子，挺有意思的，也不便宜。

某要點：

衣服和飾品最好不要都非常複雜。

像穿 CDG 可以戴珍珠，但穿 CDG 就不要戴奇怪的項鏈；

如果穿羊絨或不那麼複雜的材質，就可以戴貴金屬；

僅限模板使用，特定情況除外。
