Wittgenstein_01

10.6

  1. World（世界）  = All that is the case（所有發生的事情）  = 所有 facts（事實）
  2. 每個 fact（事實）  = 某個 state of affairs（狀況）存在  
  3. 每個 state of affairs（狀況）  = 若干 objects（事物）以某種方式組合而成
  4. objects（事物）  = 世界的基本單元，不可再分
