    一型：功能性愛（Instrumental Love）

    「你能為我帶來什麼」，包括陪伴、安全感、認可、愉悅、逃避孤獨等，所有你給我什麼，我才在意你的模式。可替代性高，依賴「行為產出」來維持關係穩定性

    二型：人格性愛（Personhood-based Love）

    「我欣賞你這個人本身，即便你給不了我什麼，甚至讓我受苦，但我依然想靠近你、對你好」，這是對“你是誰”本身的情感連結。不可替代性高，依賴「存在本身」作為價值

    1. 當愛的驅動來自於對方能「給我什麼」時 → 這段關係的穩定性依賴於持續收益 → 一型佔主導 → 替代性高。

    2. 當愛的驅動是來自於「對你這個人本身」的在意，即使你讓我痛苦、給不了我什麼，我依然願意留下 → 二型佔主導 → 替代性低。

    3. 二型越高，人的投入越接近「不計代價」的真誠；一型越高，關係越像是「交換關係」或工具式依戀。

    4. 一型關係中的人往往不能忍受對方「成為一個不好用的人」，但二型關係中的人即使對方崩壞或退化，依然有殘留的照顧/連結傾向。

    5. “一般來講，二越高，不可替代性就越高；一越高，不可替代性就越低。”；一型指向的是行為層面：誰能陪我、回我消息、提供穩定感、滿足我需求 → 可被替代；二型指向的是存有層面：「你是你」，無法從功能上定義 → 只要是你，就值得靠近 → 不可替代

    6. 基本都混著用這兩種愛的邏輯，只是比例不同，但其實按照市場邏輯來的話，現代社會所有配對機制都是基於第一型設計的，絕大多數人的感情都是第一種為主導的。比方說正常情況下，普通的談戀愛，按條件匹配的話，一般都是一比較多。 第二種的話,（不包括说依賴性、没得换、BPD、受虐狂、死活不分手等品类），那其實是緣分問題，遇上了就有，遇不上那就没有。
