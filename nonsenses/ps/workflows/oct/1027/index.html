  1. 在一切资源都被稀缺化的背景中，权力者的身体异常丰盈，正是权力结构掠夺性、压榨性、极端性的一种具象体现。

  2. something about racism：

    前提条件：

    已经取得某发达国家/白人社会的基本生活准入门槛的（特别是某些性别的）普通人 ( for example : EAD）

    “普通人”＝资源有限、无结构性特权、仅靠个人努力生存的人。

    问题：

    “在现实社会系统中，如果面对二选一的情况，一个普通人更应该尽量避免的歧视来源，是基于种族的结构性排斥，还是基于性别与年龄的结构性规训与淘汰？”

    非极端情况下：

    A : 种族歧视 => 语言攻击、职场偏见、种族冷处理

    B : 性别/年龄歧视（etc) => 熟人系统 + 本地文化 + 身体机制 + 主动边缘化 + 话语污名化（某些性别尤甚）

    被驯化的特征：

    被熟悉麻痹了从家庭到单位到公共话语系统无缝全覆盖的‘合情合理地羞辱’

    反而对陌生社会的排斥更敏感，才会只在这个时候觉得被歧视。

    熟悉的暴力会穿上"这就是生活"的外衣，陌生的偏见会被感知放大。
